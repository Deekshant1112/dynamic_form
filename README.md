# Dynamic Form Builder (EJS + MongoDB) - Final ZIP

## What you get
- Full Express app with EJS views, styled with a modern blue/glass theme.
- Admin pages: login, dashboard, create/edit, fields manager, submissions.
- Public pages: index, about, contact, form renderer.
- Server-side validation and MongoDB persistence.
- Seed script to create a sample form.

## Run
1. `.env` and set `MONGO_URI` and `ADMIN_TOKEN`.
2. npm install
3. npm run seed   # optional
4. npm start
5. Open http://localhost:4000/
